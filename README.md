# ✍️ Kuis Menulis Karakter Jepang
Handwriting Recognition menggunakan $P Point Cloud Recognizer

## Deskripsi
Aplikasi latihan menulis karakter Jepang dengan sistem pengenalan tulisan tangan yang berjalan penuh di browser tanpa backend.

## Fitur Utama
- Pengenalan tulisan tangan realtime
- Mode kuis interaktif
- Scoring otomatis berdasarkan tingkat kemiripan
- Support sentuh & mouse
- Responsive untuk mobile & desktop

## Arsitektur
- **MVC Pattern**
- 100% Client Side JavaScript
- Tanpa dependensi eksternal
- Menggunakan algoritma $P Point Cloud Recognizer
